# Экспериментальная часть

Получерновая исследовательская работа по шагам. Чистовое решение — в `../Решение.ipynb`.

- `_common.py` — загрузка данных и базовые величины сигнала.
- `01_eda.py` — структура и форма сигнала, baseline, насыщение.
- `02_features_and_psd.py` — признаки формы, подбор окна PSD, две ветви.
- `03_clustering_comparison.py` — почему «в лоб» не работает; двухэтапная схема.
- `04_anomaly_search.py` — поиск нумерации меток и доли/критерия аномалий по Kaggle.
- `05_selftrain_boundary.py` — self-training для границы классов и финальная модель.
